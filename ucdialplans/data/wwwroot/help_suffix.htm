Entering a rule suffix can help you distinguish between extension blocks for different sites.

For example: If you are entering an extension range for your head office in London, UK, you could enter a suffix like "HeadOffice". The generated rules would look like UK-London-20-HeadOffice.

Rule suffixes can contain letters, numbers, dashes or underscore characters. No spaces are allowed.

This is optional. If not entered, the Optimizer will use a suffix of "Internal" for the rule names.
