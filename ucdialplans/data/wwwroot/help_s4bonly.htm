If the extension range targets users on a legacy PBX, leave this checkbox blank. The Optimizer will create the necessary routes and trunk translation rules to send only extension digits to the upstream PBX.

If the extension range includes only Lync/Skype for Business Enterprise Voice users, select this checkbox. The Optimizer won't create unnecessary routes or trunk translation rules, which aren't required for routing calls to internal Lync/Skype for Business users.
